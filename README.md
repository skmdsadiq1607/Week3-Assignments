# Product API Project

## About
This repository contains a simple JavaScript project based on API handling and product data management. The project was created to practice fetching, displaying, and working with product-related data using JavaScript.

It helps in understanding basic API concepts and asynchronous programming in JavaScript.

---

## File Included

- `ProductAPI.js`  
  Contains the main logic for handling product API data and related operations.

---

## Technologies Used

- JavaScript
- Node.js
- API Handling Concepts
